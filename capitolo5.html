Le reti

In informatica, una rete (di computer) è un insieme di dispositivi collegati tra loro per scambiare dati e risorse. Dispositivi: computer, smartphone, stampanti, server, router… Collegamenti: via cavo (Ethernet) o senza fili (Wi-Fi) Regole di comunicazione: chiamate protocolli (es. TCP/IP La rete è importante perche puo fare:Condividere dati (es. file, documenti)
  • Usare stampanti in comune
  • Accedere a internet
  • Comunicare (es. email, videochiamate)
  • Fare backup su server o cloud
